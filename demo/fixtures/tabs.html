Overview

Lit SSR renders your LitElement components to Declarative Shadow DOM on the server, giving users instant first paint with no layout shift.

 Architecture

The pipeline: TypeScript components are bundled with esbuild, compiled to WASM via Javy (QuickJS), and executed from Go using the wazero pure-Go WASM runtime.

 Performance

Cold start ~350ms per render. The 2MB WASM module embeds QuickJS, Lit SSR, and all component definitions.

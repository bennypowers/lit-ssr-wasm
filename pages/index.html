lit-ssr-wasm

Server-side render Lit web components from any language via WebAssembly

Overview
LitElement (TS) esbuild Javy (QuickJS) WASM (2 MB) Go / wazero

How it works

Standard Lit web components are bundled with @lit-labs/ssr, compiled to WebAssembly via Javy (QuickJS), and executed from Go using wazero (pure Go, no CGo).

The result: Declarative Shadow DOM injected into your HTML on the server, giving users instant first paint with zero layout shift. No Node.js sidecar required.

Two modes

Compiled

Component definitions are baked into the WASM module at build time. Fast, predictable, ideal for cem serve or static site generators.

Runtime

Component definitions are provided as JavaScript source at render time. Flexible, dynamic, ideal for design tools or component playgrounds.
